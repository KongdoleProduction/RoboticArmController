## 사용법

회로도는 /schematics/ 폴더 안에 있습니다.
- `shield.png` 는 이미지 파일입니다.
- `shield.sch` 는 eagle cad 파일입니다.

아두이노 소스 코드는 /arduino_code/ 폴더 안에 있습니다.
- 다운로드 후 Arduio IDE로 열어서 아두이노 보드에 업로드해주세요.
- Uno용으로 설계되었지만 대부분의 다른 아두이노 보드에서도 사용 가능합니다.
